# Site Clearing Simulator

A NextJS Application for simulating vehicle movements utilizing MVVM (Model View ViewModel) architecture

## How to use
Install it and run:

```sh
npm install
npm run dev
```
## Technologies Used
- TypeScript
- React
- NextJS
- Mobx
- GraphQL
- Auth0
- PostgreSQL
- Hasura

## Node and NPM Versions
- Node: 16.15.0  
- npm: 8.5.5
